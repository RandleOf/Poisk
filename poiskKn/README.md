React приложение для отображения работы системы + api для отображения текста на сайте.
I. Инициализация api:
1. Для начала надо установить библиотеки из requirements.txt (я правда не знаю какие 100% нужны)
2. Далее надо перейти в директорию "/poiskKn/backend/..."
3. В ней через консоль "cmd" выполнить команду:
   * python api.py
4. Далее надо создать ещё одну консоль (powershell)
II. Запуск React приложения:
1. Для начала надо перейти в директорию "/poiskKn/..."
2. Далее просто надо выполнить несколтко команд:
   * npm install
   * npm run dev
3. Локальный сервер находится (вероятнее всего) по адресу "http://localhost:5173/"
